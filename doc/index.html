Installation instructions

  • Download and install latest version of NodeJS on https://nodejs.org/en/
  • Unzip delivered zip file
  • Run npm install command line in your preferred shell

Running the scraping tool

  • Run npm main.js --help for command parameters
  • Example : node main.js -c boatsGigSample.csv -b boatsResults.json -v vendorsResult.json

Interpreting results

  • During execution a progress bar shows you progress of scraping process based on number of boats in CSV file
  • Will provide a small report showing: # of boats in input file, # of boats in input file marked as already processed (if replaying an existing file) and # of processed boats in output file
  • Output file named ".csv.updated" will contain exact copy of the input file plus "Done" & "Timestamp" columns
  • Done column is either null either "Y" and can be used to identify boats that were not processed. You can use this output file as the new input file - you have to transform it from JSON to CSV (using http://www.convertcsv.com/json-to-csv.htm for example)
  • Timestamp column contains an integer representing date / time when processing of a line was done.
